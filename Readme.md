Technical Stack
Backend: PHP (8.x)

Database: MySQL

Frontend: Tailwind CSS (via Template)

Architecture: Procedural PHP with clear separation of logic (scripts/) and presentation (public/).

Evaluation Criteria Respected

Redirections: Fluid navigation using header("Location: ...").

Logic: Robust conditional structures for form validation and error handling.
